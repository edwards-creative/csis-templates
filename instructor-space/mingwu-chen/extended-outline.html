Course Description

An introduction to the creation, management, and technology of professional websites. Topics include: Page design and editing in HTML and CSS using both basic tools such as text editors and professional web development tools such as Dreamweaver; Image display and manipulation using professional graphics tools such as Adobe Photoshop or GIMP; Other topics covered include an introduction to JavaScript, mobile devices, blogs, wikis, unix basics, dynamic content generation using server side scripting.
